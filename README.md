# Navi Volo Staking Project

## 項目概述
一個展示 Navi Volo Staking 數據同支援質押嘅網站，包含後端 API 同前端展示。

## 文件結構
- `backend/`: Python 後端代碼同依賴。
- `frontend/`: HTML 前端代碼。
- `contracts/`: Move 智能合約。

## 部署
1. 後端：部署到 Render[](https://render.com)。
2. 前端：部署到 GitHub Pages。
3. 合約：用 SUI CLI 部署到 SUI 主網。
